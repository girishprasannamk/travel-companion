# Travel Companion Architecture

## Vision

Travel Companion is an offline-first Progressive Web App (PWA) designed to help users plan, travel, track expenses, manage tickets, and relive trips.

## Core Principles

- Offline First
- Mobile First
- Data Driven
- Reusable Trip Engine
- Beautiful UI
- Zero Backend (Phase 1)

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- GitHub Pages
- Progressive Web App (PWA)
